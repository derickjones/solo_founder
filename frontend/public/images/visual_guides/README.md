# Visual Guides

This folder contains weekly visual guide infographics for the Gospel Study App.

## File Naming Convention:
- `visual_guide_week_01.png` - Week 1 visual guide
- `visual_guide_week_02.png` - Week 2 visual guide
- `visual_guide_week_XX.png` - Week XX visual guide (up to 52)

## Image Specifications:
- **Format**: PNG or JPG
- **Aspect Ratio**: 16:9 (recommended for best display)
- **Resolution**: 1920x1080 or higher
- **File Size**: Keep under 2MB for optimal loading

## Usage:
The VisualGuideViewer component automatically detects which images exist and displays them with week navigation controls.

Simply drop your infographic files into this folder following the naming convention above.
